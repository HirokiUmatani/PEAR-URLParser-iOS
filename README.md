====
### Overview
This is the ios library to parse the URL.

### Installation
<code>
pod 'PEAR-URLParser-iOS'
</code>

### Constitution  
URLEntity  
└url  
└scheme  
└host  
└path  
└paths<NSArray>  
└lastPath  
└fileName  
└extention
